My personal website was sort of an experiment. I always had an idea to create a 'pin' board type layout using the Masonry framework and a few concepts later, here we are! I listen to music all the time whilst I am working and wanted a place to store music videos/remixes from Youtube once I broke the 200 video limit on a single playlist. Every video is now stored using Youtubes API into WordPress.
